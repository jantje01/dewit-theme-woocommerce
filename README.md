# Dewit Theme

A custom WordPress theme with WooCommerce integration.

## Features

- WordPress theme setup with menus, logo, thumbnails, editor styles, and `theme.json`.
- WooCommerce support for product galleries, shop wrappers, product grids, and checkout pages.
- Responsive PHP template structure for pages, posts, archives, search, 404, and shop pages.
- Lightweight CSS and JavaScript with no build step.

## Installation

1. Upload this folder to `wp-content/themes/dewit-theme`.
2. Activate **Dewit Theme** in WordPress.
3. Install and activate WooCommerce.
4. Configure menus, logo, homepage, and WooCommerce pages in the WordPress admin.

## Development

The main theme setup lives in `functions.php`. Frontend CSS and JavaScript live in `assets/`.
